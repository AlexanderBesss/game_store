## Docker

```bash
# start app in docker
$ docker-compose up
```

## Production

```bash
$ npm install

# build app
$ npm run build

# create tables in procuction db
$ npm run init:prod

# start production app
$ npm run start:prod
```

## Development

```bash
$ npm install
# create tables in development db
$ npm run init:dev

# start app
$ npm run start

# start app in watch mode
$ npm run start:dev
```
